paciam /survey puslapyje viska pritaikyti prie dabartinio stiliaus, tik pakeisti klausimus 

**1 Klausimas (Srautas)**

- **Antraštė:** Kiek vidutiniškai naujų užklausų gaunate per mėnesį?
- **Input tipas:** Skaičiaus įvedimas (Number Field).
- **Kintamasis (Variable):** LEADS
- **Mygtukas:** Toliau ➔

**2 Klausimas (Vertė)**

- **Antraštė:** Kokia yra vidutinė vieno kliento vertė?
- **Sub-tekstas:** Kiek pajamų jums atneša vienas sėkmingas sandoris?
- **Input tipas:** Skaičiaus įvedimas su € simboliu.
- **Kintamasis (Variable):** VALUE
- **Mygtukai:** Atgal ←, Toliau ➔

**3 Klausimas (Efektyvumas)**

- **Antraštė:** Koks jūsų dabartinis pardavimų procentas?
- **Sub-tekstas:** Kiek realistiškai iš 100 užklausų tampa pardavimais?
- **Input tipas:** Slankiklis (Slider) nuo 1% iki 50% (Default: 10%).
- **Kintamasis (Variable):** CLOSE_RATE (formatas: 0.10)
- **Mygtukas:** Atgal ←, Toliau ➔

**4 Klausimas (Greitis)**

- **Antraštė:** Kaip greitai jūsų komanda susisiekia su nauja užklausa?
- **Input tipas:** Pasirinkimas (Radio Buttons).
    - A) Per 5min.
    - B) Per 1val.
    - C) Tos pačios dienos bėgyje
    - D) Kitą darbo dieną arba vėliau
- **Mygtukas:** Generuoti ataskaitą ➔

### **TECHNINIS PROMPTAS (Developers/AI Builders)**

Jei naudosite AI įrankį ar programuotoją, nukopijuokite šią logiką, kad skaičiai veiktų teisingai:

code JavaScript

`// KINTAMIEJI IŠ SURVEY
let leads = input.leads; // Pvz: 100
let value = input.value; // Pvz: 1000
let close_rate = input.close_rate / 100; // Pvz: 0.10

// 1. DABARTINĖS PAJAMOS
let sales_now = Math.round(leads * close_rate);
let rev_now = sales_now * value;

// 2. POVEIKIS 1: GREITIS (SPEED BOOST +4%)
let boost_speed = 0.04;
let sales_speed = Math.round(leads * boost_speed);
let rev_speed = sales_speed * value;

// 3. POVEIKIS 2: REAKTYVACIJA (NURTURE +2-3% nuo likusių)
// Skaičiuojame likusius leadus, kurie nepirko nei standartiškai, nei dėl greičio
let remaining_leads = leads - (sales_now + sales_speed);
let boost_nurture = 0.025;
let sales_nurture = Math.round(remaining_leads * boost_nurture);
let rev_nurture = sales_nurture * value;

// 4. TOTAL SU AKSELER
let sales_total = sales_now + sales_speed + sales_nurture;
let rev_total = sales_total * value;
let new_close_rate = Math.round((sales_total / leads) * 100);

// 5. PRARASTI PINIGAI (BOX 1 REZULTATAS)
let money_lost = rev_total - rev_now;
let money_lost_year = money_lost * 12;

// 6. LAIKO SUTAUPYMAS (BOX 2 SKAIČIAVIMAI)
// Prielaida: 10 min kvalifikavimui, 15 min follow-up (90% leadų reikia follow-up)
let time_qualify_hours = Math.round((leads * 10) / 60);
let time_followup_hours = Math.round((leads * 0.9 * 15) / 60);
let total_time = time_qualify_hours + time_followup_hours;
let days_saved = (total_time / 40).toFixed(1); // Lyginama su 40val darbo savaite

// FORMATAVIMAS
// Visi piniginiai skaičiai turi būti atvaizduoti su kableliais (pvz. 10,000)`
